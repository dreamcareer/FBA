# デプロイ前チェック

デプロイ前に以下を順番に確認してください:

1. **Lint チェック**
   ```bash
   npm run lint
   ```

2. **ビルドチェック**
   ```bash
   npm run build
   ```

3. **Prisma スキーマ検証**
   ```bash
   npx prisma validate
   ```

4. **環境変数の確認**
   - `.env` に必要な変数がすべて設定されているか確認
   - `NEXT_PUBLIC_` 以外の変数がクライアントコードで参照されていないか確認

5. **未コミットの変更確認**
   ```bash
   git status
   ```

すべてパスしたら「デプロイ準備完了」と報告してください。
問題があれば内容と修正案を提示してください。
