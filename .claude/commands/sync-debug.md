# 同期デバッグ

ロジレス同期に関する問題を調査してください:

1. **OAuth トークンの状態確認**
   - `src/lib/logiless/client.ts` のトークンリフレッシュロジックを確認
   - DB の `OAuthToken` テーブルの有効期限を確認

2. **同期ログの確認**
   - `sync_logs` テーブルの最新のエラーを確認
   - エラーメッセージからAPIレスポンスの問題を特定

3. **API呼び出しの確認**
   - リクエストパラメータが正しいか
   - レート制限（429）に引っかかっていないか
   - リトライロジックが正しく動作しているか

4. **データマッピングの確認**
   - ロジレスのレスポンスフィールドとPrismaモデルの対応が正しいか

問題箇所を特定し、修正案を提示してください。
